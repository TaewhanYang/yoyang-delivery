# 🏥 요양원 배송 앱 (직원용)

React Native + Expo 기반으로 만든 약국 직원용 내부 배송 앱입니다.

---

## 📦 기능 요약

- 직원 이름 입력
- 요양원 검색 및 배송 루트 추가
- 배송 완료 체크 (메모 & 사진 가능)
- 운행 거리 수기 입력 + 인증 사진
- Firebase 연동

---
